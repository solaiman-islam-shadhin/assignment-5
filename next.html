Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Retrieves or sets the HTML content of an element, including HTML tags.
innerText: Retrieves or sets the text content of an element, excluding hidden elements.
textContent: Retrieves or sets the text content of an element, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in which a single event listener is added to a parent element to manage events for all its child elements. This technique leverages event bubbling, where events propagate up the DOM tree. Event delegation helps improve performance and manage events efficiently.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process in which an event triggered on an element propagates (or "bubbles") up through its ancestors in the DOM tree. This means that the event is first captured and handled by the target element, then it bubbles up to its parent, and so on, until it reaches the root of the DOM.>
